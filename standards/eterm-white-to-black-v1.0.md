====================================================
Jimmy 专属 Eterm 白转黑标准操作模板（v1.0）
====================================================

【适用范围】
- 国际 / 国内 白转黑订位
- 散客 / 家庭 / 项目组统一操作

【核心原则】
- 人工只负责：查航班 + 手工 SS 锁航段
- 其余定位编号所需结构：全部按本模板补齐
- 不简化、不省略、不使用系统默认

----------------------------------------------------
一、整体工作流程（强制统一）
----------------------------------------------------

Step 1｜人工操作（只做这一件事）
- AVH 查询航班
- 手工选择最合适航班
- 手工 SS 锁航段

注意：
- SS 是唯一人工核心步骤
- 后续任何生成不得修改航段

----------------------------------------------------
二、必须补齐的结构模块（顺序固定）
----------------------------------------------------

1. NM   （姓名结构）
2. TKTL （出票时限）
3. SSR DOCS（证件）
4. SSR CHLD / INFT（如适用）
5. SSR FQTV（如有）
6. OSI（联系方式）

----------------------------------------------------
三、姓名（NM）规则
----------------------------------------------------

1）姓名来源
- 严格以护照首页为准
- 不使用连字符 “-”
- 复姓直接连写

示例：
正确：DODETLALETINA
错误：DODET-LALETINA

2）NM 标准写法
NM1SURNAME/GIVEN NAMES1SURNAME/GIVEN NAMES CHD1SURNAME/GIVEN NAMES

规则：
- 所有旅客一次性 NM
- CHD 仅在 NM 行标注
- INF 不单独 NM

----------------------------------------------------
四、TKTL（固定格式 · 必须统一）
----------------------------------------------------

唯一允许格式：
TKTL/0000/DDMMM/SHA497

规则说明：
- 0000   ：固定 0 点
- DDMMM ：取第一程出发日期
- SHA497：固定 Office 号

禁止：
- 使用其他时间
- 使用其他 Office 号
- 重复 TKTL

----------------------------------------------------
五、证件信息（SSR DOCS）
----------------------------------------------------

基本规则：
- 每位旅客单独一条 DOCS
- 严格绑定 P 位
- 不合并、不简写

标准结构：
SSR DOCS XX HK1 P/国籍/证件号/国籍/DOB/性别/有效期/姓/名/P?

示例：
SSR DOCS HO HK1 P/FRA/22EA48562/FRA/19SEP78/F/26JUN32/LALETINA/MARIA/P1

----------------------------------------------------
六、儿童 / 婴儿规则
----------------------------------------------------

CHD（儿童）
- 出行时未满 12 岁
- 必须写：
SSR CHLD XX HK1 DDMMMYY/P?

INF（婴儿）
- 出行时未满 2 岁
- 操作顺序必须为：
1. XN/IN
2. 去程 SSR INFT
3. 回程 SSR INFT

----------------------------------------------------
七、常旅客号（FQTV）
----------------------------------------------------

规则：
- 有则写，无则不写
- 严格按 P 位绑定

格式：
SSR FQTV TK HK1 会员号/P?

----------------------------------------------------
八、联系方式（OSI）——统一极简规则
----------------------------------------------------

固定写法（所有 PNR 统一）：

OSI HO CTCT13917313678
OSI HO CTCM18601690221/P1

规则说明：
- 仅写 1 个 CTCT
- 仅写 1 个 CTCM
- 永远绑定 P1
- 旅客再多也不增加

----------------------------------------------------
九、出票前 10 秒自检清单
----------------------------------------------------

[ ] NM 姓名与护照一致
[ ] ADT / CHD 身份正确
[ ] SSR DOCS 全部 HK
[ ] TKTL 格式正确
[ ] OSI 已存在
[ ] 无重复字段
